# Run and deploy

1. **Install NPM**
   `npm install`

2. **Deploy** (using Vercel)
   `vercel --prod`

## Run Locally

**Prerequisites:** Node.js

1. **Install dependencies**
   `npm install`

2. **Start dev server**
   `npm run dev`
