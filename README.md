# Run and deploy

1. Install NPM
 `npm install`
2. Deploy using your preferred method, here we'll use Vercel.
 `vercel --prod`

## Run Locally

**Prerequisites:**  Node.js


1. Install NPM (again here):
   `npm install`
2. Run the app:
   `npm run dev`
